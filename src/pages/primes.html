Patterns in Prime Numbers

How it works

Before starting, I developed an algorithm along the help of Bruno Astrolino E Silva1 to print out prime numbers to a .txt file. In this case, I calculated roughly all primes under \(10^8\).

			
				def primes(n):
   					sieve = [True] * n
    				for i in range(3,int(n**0.5)+1,2):
        				if sieve[i]:
           					sieve[i*i::2*i]=[False]*((n-i*i-1)//(2*i)+1)
    					return [2] + [i for i in range(3,n,2) if sieve[i]]

				num_primes = int(input("How many primes: "))
				p = primes(num_primes)
				formatted = "".join(str(p))

				file = open("primes.txt", "a")
				file.write(formatted)
				file.close()
